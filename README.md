# 724 Events

## Description
L'application est le site d'une agence evenementielle.
## Pre-requis
- NodeJS  >= v16.14.1

## Installation
- `yarn install`

## Lancement de l'application
- `yarn start`

## Tests
- `yarn test`

- Vous êtes développeur front-end freelance, et l’agence évènementielle 724events vous contacte pour une mission.
